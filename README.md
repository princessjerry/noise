# 난청 아동의 소음 환경 기능적 의사소통 평가 프로토타입

## 프로그램 개요

난청 아동의 소음 환경 기능적 의사소통 반응을 기록하기 위한 웹 기반 평가 보조 프로토타입입니다. 표준화 검사 또는 진단 도구가 아니며, 결과를 진단 목적으로 단독 사용해서는 안 됩니다.

## 실행 방법

정적 웹 서버 또는 GitHub Pages에서 실행합니다. `index.html`을 직접 열 수도 있지만, 브라우저 보안 정책에 따라 오디오 재생이 제한될 수 있습니다.

## 필요한 파일

- `index.html`
- `classroom_noise.mp4`
- `item01.mp3` ～ `item18.mp3`

## 음성 파일 규칙

- 문항 번호와 음성 파일명이 일치해야 합니다.
- 모든 문항은 동일한 녹음 또는 음성 제작 조건을 사용하는 것이 권장됩니다.
- 음성 파일 자체는 이번 코드 수정에서 생성하거나 편집하지 않았습니다.

## 주의사항

- 화면상의 재생 설정은 실제 음압 또는 표준화된 신호대잡음비를 의미하지 않습니다.
- 실제 출력은 사용하는 기기와 시스템 음량에 따라 달라질 수 있습니다.
- 결과는 보호자 보고, 임상가 관찰 및 추가 평가와 함께 검토해야 합니다.
