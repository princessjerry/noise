평가를 시작하기 전에 목표 음향 조건, 보정 프로필, 장비 조건 및 시험음 재생 상태를 확인합니다.

1. 목표 음향 조건

목표값
표시된 수치는 연구에서 설정한 목표 음향 조건입니다. 실제 음향 조건은 지정된 재생 장비와 수검자 위치에서 소음계를 이용하여 측정한 값을 기준으로 합니다.

2. 보정 프로필 선택

미보정
상세정보 보기
위 값은 본 연구에서 설정한 보정 허용범위입니다. 초기 기준은 문장 59.0~61.0 dBA, 소음 54.0~56.0 dBA, SNR +4.0~+6.0 dB입니다.

예를 들어 특정 노트북의 컴퓨터 음량 38%에서 문장 수준이 60 dBA로 측정되었다면, 38%라는 설정은 해당 노트북, 외장 스피커, 스피커 출력, 재생 위치 및 측정 거리에서만 유효합니다.

컴퓨터 음량의 백분율은 실제 dBA를 의미하지 않습니다.

노트북, 스피커, 오디오 출력 장치, 스피커 출력 다이얼, 거리, 방향, 검사 장소, 음원 파일 또는 웹 재생 설정이 변경된 경우 음압을 다시 측정해야 합니다.

    보정 프로필 관리

    관리자 또는 연구책임자가 음향 보정 후 등록하는 화면입니다. dBA 값은 외부 소음계로 측정한 값을 직접 입력하며, 브라우저 음량값이나 마이크로 자동 추정하지 않습니다.

    측정된 문장 수준과 소음 수준을 입력하면 SNR이 자동 계산됩니다. 값이 하나라도 없으면 ‘미계산’으로 표시됩니다.
